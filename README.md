# Gabriel Polastrini - Test Vaga Mobile

Fiz o Aplicativo em React Native, usando React Navigation, React Hooks, PokeAPI e as imagens do site oficial do Pokemon.

## Demo

![Demo](./desafioPokemon/assets/demo.gif)

## Instruções

1. Clone o repositório
	`git clone https://https://github.com/0xbiel/vaga-mobile-teste/`

2. Instale as dependencias
	`npm install`

3. Inicie o servidor do react native
	`npx react-native start`

4. Inicie o aplicativo no Android
	`npx react-native run-android`
